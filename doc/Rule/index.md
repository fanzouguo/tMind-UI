# 组件规范

## 通用属性
|      名称     |    说明     |    类型    |    可选    |  默认值  |
|    :-----:    | :-----:    | :----:     | :----:     | :----:  |
|      config   |    组件参数集合     |    object    |    参考config定义    |  config定义 |

## config定义
|      名称     |    说明     |    类型    |    可选    |  默认值  |
|    :-----:    | :-----:    | :----:     | :----:     | :----:  |
|      title    |    显示标题     |    string    |    组件约定    |  组件约定  |
|      tWidth   |    宽度     |    string \| number(数字型默认转换为px单位)    |    组件约定    |  组件约定 |
|      tHeight   |    高度     |    string \| number(数字型默认转换为px单位)    |    组件约定    |  组件约定 |
|      tClass   |    自定义样式     |    object    |    组件约定    |  组件约定 |
